Features

Everything your chatbot needs

01

Visual canvas.
Drag, connect, publish.

The WAgen Flow Builder gives you a canvas where every conversation step is a visible node. Add messages, ask questions, show buttons, set conditions, trigger actions — all by dragging and connecting blocks. See the entire conversation tree at once, spot gaps, and fix them before going live.

  • Message nodes: text, image, video, document, catalog
  • Input nodes: open question, button choice, list picker
  • Condition nodes: if/else branching on any field value
  • Action nodes: tag contact, assign agent, send to webhook
  • Delay node: pause X minutes/hours before next step
  • Fallback node: handle unexpected replies gracefully

FAQ

Common questions

Do I need coding skills to build a flow? +
No. The WAgen Flow Builder is 100% visual. You drag blocks onto a canvas and connect them with arrows. If you can draw a flowchart on paper, you can build a WAgen flow. There is no HTML, JavaScript, or API knowledge needed.
Can my chatbot handle free-text replies, not just button taps? +
Yes. WAgen flows support both structured replies (buttons, list pickers) and open text input. For open text, you can use keyword matching (contains "refund", starts with "ORDER") or capture the raw input as a contact field for your team to review.
What happens if a customer asks something my flow doesn't cover? +
You configure a Fallback node: a friendly "I didn't catch that" reply with an option to restart, pick from the main menu, or escalate to a human agent. You also see all unmatched queries in the analytics dashboard — these are your top candidates for new flow branches.
Can one conversation trigger multiple flows? +
Each active keyword/trigger can only point to one flow at a time. But a flow can call another flow as a sub-flow, or change the active context so the next message triggers a different flow. Complex multi-path experiences are fully supported.
Can I run a flow for existing contacts via broadcast? +
Yes. When you send a broadcast message with a button, tapping that button can trigger a specific flow. This is how retargeting and re-engagement campaigns work — send a broadcast to 500 past customers, and those who tap "Yes, interested" enter a qualification flow automatically.
Does the Flow Builder work with WhatsApp Business API (not the app)? +
Yes, WAgen Flows work exclusively over the official WhatsApp Business API (Meta WABA). This is different from the regular WhatsApp app or WhatsApp Business App. All automated messages are template-based or within the 24-hour service window, fully compliant with Meta's policies.
