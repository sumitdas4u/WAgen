How It Works

Trigger. Wait. Send. Repeat.

Sequences run entirely on autopilot — from the moment a contact enters to the final message in the series.

①

Trigger on contact event

A new contact is created (from WhatsApp, QR, widget, or API) or an existing contact's profile is updated. WAgen evaluates your start conditions.

②

Enrol matching contacts

If the contact matches your audience filter — type, tags, custom fields — they're enrolled in the sequence. Existing contacts enrolled at step 1.

③

Messages sent on schedule

Each step fires at its configured delay. Step 1 immediately, step 2 after 2 hours, step 3 after 3 days. Delivery windows keep messages within business hours.

Start and stop conditions.
Smart enrolment, zero spam.

Not every contact should enter every sequence. Define start conditions to filter who gets enrolled and stop conditions to exit contacts when they've converted — so you never send an offer to someone who already bought.

  • Start conditions: contact type, tags, custom field values
  • Stop conditions: "tag added = purchased", "field set = booked"
  • Allow-once mode: each contact enrolled maximum once per sequence
  • Require previous delivery: only advance if last message was delivered

02

Delivery windows.
Messages only when it matters.

Nobody wants a 2 AM WhatsApp message. Set allowed days (weekdays only, or include weekends) and a delivery window (e.g. 9 AM – 7 PM). If a step is scheduled outside the window, WAgen holds it until the next allowed slot.

  • Set allowed days: Mon–Fri, Mon–Sat, or every day
  • Set time window: e.g. 9:00 AM to 7:00 PM in your timezone
  • Messages held until next allowed slot — never dropped
  • Per-sequence settings — B2B sequences can use stricter windows

FAQ

Common questions

Can a contact be in multiple sequences at once? +
Yes. A contact can be enrolled in multiple sequences simultaneously — for example, a "welcome" sequence and a "product launch" sequence. Each sequence runs independently on its own schedule.
What happens if a message fails to deliver? +
If you enable Require previous delivery, WAgen will retry the step for up to the configured retry window. If delivery still fails, the contact is marked as failed for that sequence. With retry disabled, the sequence continues to the next step regardless.
Can I stop a contact from a sequence if they reply "STOP"? +
Yes. You can set a stop condition that monitors for specific inbound messages (e.g. message contains "STOP") and immediately removes the contact from the sequence. This is also how opt-out compliance works.
Does sequence sending count towards my daily message limit? +
Yes. All WhatsApp template messages — whether from sequences, broadcasts, or reminders — count towards your Meta account's daily message tier. WAgen distributes sequence sends across the day to stay within limits.
